Secret Orginazation Database With Java UI

Last Modified: 10/13/2024

"Secret Orginazation Database" was a two month long project, where me and a partner in my graduate Databases Class. This large complex project is a full database, paired with a java GUI for ease of use. It showcases many technical features such as:
  • Full Relational Database
  • Java User Interface
  • Remote Database Communication
  • Complex Queries
  • Views to restrict access for specific users
  • Users can Update Missions, Equipment Use, and an Agent Status
